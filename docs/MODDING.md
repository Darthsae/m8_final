## Items
Items use the following structure:

```json
{
  "name": "Display Name",
  "description": "Display Description",
  "tags": [
    "List",
    "Of",
    "Tags"
  ],
  "stack": 24,
  "uses": []
}
```

## Entities
Entities use the following structure:

```json
{
  "name": "Display Name",
  "description": "Display Description",
  "tags": [
    "List",
    "Of",
    "Tags"
  ],
  "hp": 24,
  "xp": 5
}
```

## Classes
Classes use the following structure:

```json
{
  "name": "Display Name",
  "description": "Display Description",
  "level_data": []
}
```
